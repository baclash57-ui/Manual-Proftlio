**Overview**

The Road to QA is a private learning platform designed to help users learn and practice Quality Assurance skills. The platform provides structured learning, hands-on practice, and community interaction with mentors and fellow learners. Access to the platform is granted after payment.

**Core User Flows**

Enrolling in a QA course

Viewing reviews and user successstories

Interacting with mentors and community members.

Participating in live classes and Viewing recorded classes 

**In-Scope Areas**

User journey from the homepage to the payment section

Functional links on the homepage

Videos and images displayed in the user stories section

**Out-of-Scope Areas**

Links located in the footer section
