## Overview

LInk:

https://www.candystore.com/

This is an e commerce site that focuses on purchase and distribution of different kinds of candy, The primary user flow is is browsing, searching, adding products to cart and completing checkout.

A major issue would from any of these could affect the user experience and the business at large.

### Critical user flows include:

1. search for a product
2. Filter Options
3. Browsing for products by category
4. Adding items to cart
5. Proceeding to checkout
6. paying for the items in cart
7. viewing FAQs

## In scope Areas:

1. Home page navigation
2. Product discovery (search, filter, browse products) 
3. Cart section
4. User interaction and User experience

## Out of Scope Areas:

1. Payment section
2. Admin features
3. minor issues ie minor misspelling , minor overlay of texts 
4. More information section (except FAQs)
5. About Us section
